实例

[!-- 这里我用的是C++代码块进行显示C语言，因为C++是C的一个超集，C++可以包含部分C语言 --]

[[code type="Cpp"]]

#include <stdio.h>

//头文件

 

int main()//主函数

{

    //输出字符“艹”，并且换行

    printf("艹 \n");

 

    return 0;//正常退出

}

[[/code]]

输出结果：

以下为输出结果

艹

以上为输出结果

所有的 C 语言程序都需要包含 main() 函数，代码从 main() 函数开始执行

/* ... */ 用于注释说明，运行时会被程序忽略

printf() 用于格式化输出到屏幕，printf() 函数在 "stdio.h" 头文件中声明，括号里的文字会显示出来

stdio.h 是一个头文件 (标准输入输出头文件) , #include 是一个预处理命令，用来引入头文件， 当编译器遇到 printf() 函数时，如果没有找到 stdio.h 头文件，会发生编译错误

return 0; 语句用于表示退出程序
